COMO USAR A BUILD

Oboro é um mago bom em eliminar alvos importantes, procure boas oportunidades e use seu combo para eliminar as fontes de dano do time inimigo.

O auto ataque de Oboro é mais forte que o de todos os outros magos(a curto alcance). Abuse disso na lane phase para causar dano extra no inimigo, usando o gancho para ficar próximo dele.

Oboro não é muito efetiva em teamfights. Fique usando sua primeira habilidade para pokear e, quando surgir uma boa oportunidade, use seu gancho para se aproximar e ulte para eliminar um alvo importante. O cenário ideal para Oboro é encontrar um alvo isolado e eliminá-lo com seu combo. Aproxime-se com a 2, use a 1, dê hits básicos a curto alcance e ulte se necessário.

Ordem de upgrade das habilidades: 3ª → 1ª → 2ª.
