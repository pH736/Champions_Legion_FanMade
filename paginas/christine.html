COMO USAR A BUILD

Como jungler, você deve se manter sempre farmando para ficar forte, dar bons ganks e derrotar monstros épicos para garantir buffs para a equipe. Como Christine, é importante pegar alvos isolados para não espalhar a ultimate, além de ficar longe de gramas, minions e monstros épicos quando desejar abater um inimigo. Quando for derrotar um monstro épico, faça-o sempre com um aliado próximo, para que você possa ultar sem resetar o HP do monstro.

Use a segunda habilidade para mobilidade, sustein e para carregar a barra de fé. Além disso, use-a sempre antes de atacar algum alvo para dobrar o dano do seu ataque básico. Avance com a 1ª habilidade, use os ataques básicos melhorados pela 2ª habilidade e, se necessário, use a ultimate para finalizar. Christine é ótima para eliminar inimigos na back line, como por exemplo um Hagen que esteja causando problemas para seu time. Saber dar um bom pickoff ou um bom abate na back line é crucial.

Ordem de compra dos itens: Inicie com a faca de caça(), compre a lâmina de caçador(), compre as botas fortes, a lâmina brilhante, sede de sangue, feche o chifre do rei da selva e depois pode buildar os itens na ordem exibida acima.

Ordem de upgrade das habilidades: 3ª → 1ª → 2ª.
